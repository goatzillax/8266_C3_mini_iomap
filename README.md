IO mapping notes going from Lolin ESP8266 to ESP32 C3 Mini.

| -------- | --------------------- | ---------- | -------- | ----------------------- | ---------- | -------- | ----- |
| I2C SCL  | GPIO5/SCL/D1          |            |          | IO10                    |            | IE       | Used by OLED Shield |
